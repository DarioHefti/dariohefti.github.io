Google Timeline Data Converter

Convert your json to gpx

This is an open source google timeline to gpx converter.
Everything happens on the frontend, all your data stays with you!
You need to upload your json file which google gave you.
If you choose a start and end date,
then everything outside these dates will be ignored.
If you specify "Google Place Details" API-Key,
then a request for each waypoint will be made.
If you don't have an API Key for "Google Place Details",
then you can get one here.

If you use an API-Key then the request will go through https://corsproxy.io
if this bothers you then please do NOT fill out the API-Key field!
